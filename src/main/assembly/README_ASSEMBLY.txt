La distribución de ejemploViafirma 2.14.1 contiene lo siguiente:

- El fichero ejemploViafirma.war con el binario de la aplicación de ejemplo empaquetado
 para Tomcat 6/7 y preconfigurado para su funcionamiento desde http://localhost:8080/ejemploViafirma

- El directorio sources con todos los fuentes del proyecto de ejemplo en formato maven.

- El directorio dependencias con todas las librerias dependientes del proyecto.
  Nota: Si se  añade la dependencia maven al proyecto, no es necesario añadir manualmente las dependencias.

Instrucciones para la ejecución e instalación.
----------------------------------------------
 1.-Copiar viafirma.war al directorio webapps de Tomcat (TOMCAT_HOME/webapps)
 2.-Iniciar Tomcat. (TOMCAT_HOME/bin/catalina.sh run)
 3.-Acceder con el navegador a http://localhost:8080/ejemploViafirma
